# CloudFormation Deployment (Planned)

This directory is reserved for a future CloudFormation template that will provision
the full AWS infrastructure for this project, including:

- S3 static website hosting
- API Gateway
- Lambda functions
- SNS/SQS
- DynamoDB
- IAM roles and policies

CloudFormation is listed as the primary IaC approach for this project, but the template
has not yet been implemented.

Status: Planned
